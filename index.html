About This Page

This page is hosted and maintained by the High Point University Game Design department. We have designed this page for game design students who are curious about industry roles and searching for entry-level positions or internships in the industry. We are continuously adding to this page, so if you have suggestions please direct them to Prof. Heagney at bheagney "AT" highpoint DOT edu. For now, please note the following ways to use this page:

  • Every game design role has links to Entry-Level and Internship searches for LinkedIn. For LinkedIn searches to work properly you must have a LinkedIn account and be logged in.

  • Every "game design" area contains strategies for becoming more immersed in that area. Some of the strategies are directly related to High Point University and may not be relevant to students from other programs.

Strategies

  • Double major in an area that you're interested in that will support your game design interests:
    • If you are interested in programming and engineering, consider double majoring in Computer Science
    • If you are interested in level design , consider double majoring in Interior Design
    • If you are interested in economy design , consider double majoring in Finance
    • If you are interested in narrative design , consider double majoring in English (Writing)
  • Become an active participant HPU's GAMbassador program
  • Learn to use many different game engines such as Unreal Engine, Unity, Stencyl, GameMaker, Construct, etc.
  • Develop a strong understanding of the cycle of prototyping, play testing, and iteration
  • Learn basic game programming skills using visual scripting methods (such as Blueprints)
  • Take programming classes to increase your understanding and proficiency in game logic and programming languages
  • Participate in game jams and game design competitions
  • Develop a broad skill set but specialize in one area of game design/development
  • Develop an online game design portfolio using Behance, Wix, or any other online portfolio site
  • Join University student clubs relevant to your major (Video Game Development Club, Video Game Club, etc)
  • Join International Game Developers Association (IGDA) or other relevant professional organization(s)
  • Attend Game Dev, Game Engine, or VR Meetups
  • Make, finish, and publish games

Strategies

  • Learn programming language of your choice (C#, C++, Java, Python, Maxscript, MEL, etc) and begin making tools with it
  • Study, learn, and build Special effects in game engines
  • Study, learn, and develop your own shaders in game engines
  • Fall in love with developing tools in game engines using scripts and visual scripting (blueprints).
